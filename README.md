В системе сохраняется общий реестр доверенных учреждений, однако в публичную часть доказательства не передается ни сам список, ни идентификатор учреждения, ни сырой корень Merkle-дерева. Вместо этого используется публичный commitment к состоянию реестра. Это позволяет verifier проверить, что credential выдан одним из доверенных учреждений, но не позволяет определить, каким именно учреждением
